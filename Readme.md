<A Study on Inventory Management System(IMS) Stock In&Out with /ReorderAlert in Basic Shops> - is not just a project — it’s our survival kit for final year 😄.

## Built using React + Sass for a smooth and stylish frontend, powered by Node.js & Express for backend logic, and secured with MongoDB Atlas (Cloud Database), this system manages products, categories, staff, and operations efficiently.
